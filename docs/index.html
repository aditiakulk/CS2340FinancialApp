Technical Architecture

MVVM & Design Patterns

Our app follows the MVVM architecture to decouple UI from logic. We also utilize various design patterns that we learned over the semester. For example, the Singleton pattern for single instance access to Firebase and Firestore.

MVVM Singleton Architecture

Local AI Integration

Unlike other apps, we locally run a Local Llama 3.2 model for the chatbot. This ensures financial queries are processed privately without external API dependencies. Feel free to talk to the chatbot about your deepest, darkest insecurities and worries.

Llama 3.2 Privacy First Gippity

Performance UI

We use highly optimized RecyclerViews with custom Adapters to render extensive lists of transaction history and budget items without UI lag.

RecyclerView Adapters

Backend & Sync

Firebase Firestore handles data in real-time, syncing user budgets and allows users to collborate across Savings Circles with ease.

Firebase Firestore

Application Tour

A walkthrough of the key modules in SpendWise.

Visual Dashboard

Budgeting doesn't need to be hard

Our dashboard aggregates all your financial data into clear, interactive visualizations using MPAndroidChart.

  • Pie Chart: Breakdown of spending by category
  • Bar Chart: Compare budget limits vs actual spending.
  • Calendar: Switch between dates to see historical data.

Comprehensive Expense Log

Every penny accounted for

The Expense Log uses a RecyclerView to handle unlimited transaction entries.

  • Add: Use the floating action button to log expenses
  • Details: See amounts, categories, and dates at a glance.
  • Sorting: Automatically sorted by date for easy tracking.

Smart AI Assistant

Your personal financial advisor, running locally.

Chat with our integrated AI to get insights without leaving the app. Powered by Llama 3.2.

  • Financial Overviews: Ask "Give me an overview" to get a summary of your income vs. expenses.
  • Budgeting Tips: Get personalized advice on how to save more.
  • Context Aware: The bot understands your current budget context.

Profile & Customization

Make your profile picture a cute animal

SpendWise has accounts, and they can be customized!

  • Avatars: Choose from a variety of fun 3D avatars (Dinos, Foxes, Rabbits) to represent your profile in Savings Circles.
  • Stats: View your total expenses logged and budgets created in the Profile tab.
